Week 9: October 20 - October 26

What did you do last week?

  • Worked on a take-home classification predictive model for an interview.
  • Worked on my regression model:
    • Tried a simple linear regression model, but the results were disappointing with bad mean squared error (MSE) and R-squared scores.
    • Performed correlation tests and dropped several features, but it didn’t improve performance.
    • Implemented Lasso and Ridge regularization, followed by XGBoost and Random Forest models, but still saw no improvement.
    • Realized the target variable had too much variance, making it unsuitable for regression.
    • Attempted using different continuous target variables, but the model fits were still bad, so I decided to go back to the drawing board.

What do you plan to do this week?

  • Research datasets on Kaggle to identify those that are well-suited for linear regression, then begin data cleaning and exploratory data analysis (EDA).
  • Practice behavioral interview questions.

Are there any impediments in your way?

  • Given the issues with my last dataset, I want to avoid similar problems. I plan to take my time selecting a dataset that has had prior success with linear regression on Kaggle.

Reflection on the process you used last week, how can you make the process work better?

I now realize that I overlooked some warning signs during my exploratory data analysis (EDA), especially the high variance in my target variable, because I really wanted to use this dataset. In the future, I’ll be more mindful of EDA results and proceed with caution to avoid similar mistakes.
