Vision Statement

This project aims to develop a predictive model based on human behavior. The primary goals are to enhance my resume in pursuit of a data science role, refresh my skills in dataset preparation, and learn new modeling techniques. I am particularly interested in exploring human behavior, a topic I have not previously examined in depth. Ultimately, I hope to have a project that I can confidently discuss during interviews, showcasing my proficiency with two prominent machine learning Python packages: Scikit-Learn and PyTorch.

Motivation: Why is this project important to you?

This project is important to me because it focuses on a predictive model that genuinely interests me, beyond the scope of class assignments. I believe that having a personal project to speak passionately about will significantly benefit my performance in interviews.

Specific and Measurable Goals

  • Clean and prepare the dataset.
  • Analyze the dataset to identify key variables that contribute to the model.
  • Select 2 to 3 models that best suit the dataset.
  • Create testing and training datasets if necessary.
  • Train the model and adjust variables to achieve high accuracy.
  • Develop a more advanced model using neural networks.
  • Fine-tune the neural network model to improve accuracy.

Risks to Project Completion

  • Limited experience with neural networks.
  • Potential skill gaps due to a year-long hiatus from machine learning.
  • Risk of choosing unsuitable models for the dataset, leading to potential rework.
  • Lack of experience with PyTorch.

Mitigation Strategies for Risks

  • Review the PyTorch documentation thoroughly.
  • Revisit past machine learning projects to refresh my skills.
  • Seek guidance from my data science professor for additional support.

Project Assessments: Evaluation Criteria

  • The dataset is thoroughly cleaned and prepared.
  • A detailed analysis of the dataset is completed.
  • At least 2-3 models are developed with an accuracy of 85% or higher.
  • A neural network model is created and demonstrates decent accuracy.
